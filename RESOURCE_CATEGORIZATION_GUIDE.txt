UPLOAD INSTRUCTIONS
===================

1. Go to http://localhost:8080/admin
2. Click on "Resources" tab
3. Select the appropriate sub-tab (Templates, IP 101 Tutorials, etc.)
4. Click "Add [Category]" button
5. Fill in the form:
   - Title: Use a descriptive name (e.g., "Deed of Assignment Template")
   - Category: Select from dropdown
   - Resource Type: Select appropriate type (download, guide, video, link)
   - Description: Brief description of the document
   - Upload File: Select the file from your computer
6. Click "Add Resource" to save

FILE NAMING RECOMMENDATIONS
===========================

When uploading, use clear, descriptive titles:
- "Deed of Assignment Template (2026)" instead of "DEED OF ASSIGNMENT_2026"
- "IPOPHL Trademark Application Form (Form 100)" instead of "IPOPHLFillableForm100"
- "Patent Search Strategies Guide" instead of "Patent-Search-Strategies-UPDATED"

This helps users understand what they are downloading.

RESOURCE TYPE EXPLANATIONS
==========================

- article: Written articles or blog posts
- guide: Instructional guides, manuals, or educational documents
- video: Video tutorials or recorded presentations
- download: Files meant to be downloaded (forms, templates, documents)
- link: External links to other websites or resources
